cmd commands

dir : Displays all the contents in the present directory. Listing of all contents in the particular directory.

cd : To change to a specific directory

cd.. : To go back one previous directory

cd\ : To go to the root of the directory

*To change between Drives(C:,D:,E: etc) : Type a Drive letter (Ex: "C:" or "D:") and press Enter

cls : Clear Screen

Git Commands in CMD 

git init : To create a git folder

git status : 

git remote add <alias name> <git acc link>
Eg: git remote add origin <link>

git remote -v (where 'v' stands for verbose meaning 'Interactive')

git add .

git commit -m <"Meaningful comment">

git push <alias name> master
Eg: git push origin master
